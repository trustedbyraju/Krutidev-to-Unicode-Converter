## 🛠️ Deployment (GitHub Pages)

1. Fork or clone this repository
2. Go to **Settings → Pages**
3. Set source to **Deploy from a branch** → `main` → `/ (root)`
4. Save — your site will be live in ~60 seconds

---

## 🎨 Adding PWA Icons

Create an `icons/` folder and add:
- `icon-192.png` (192×192px)
- `icon-512.png` (512×512px)

Use any online tool (e.g., realfavicongenerator.net) to generate these from your logo.

## 🔧 Customization

All content sections are clearly commented in `index.html`:
- `<!-- HERO SECTION -->` — Change headline and subtitle
- `<!-- TOOL SECTION -->` — The converter (JS logic below)
- `<!-- FEATURES SECTION -->` — Edit feature cards
- `<!-- FAQ SECTION -->` — Add/remove FAQ items
- `<!-- ABOUT SECTION -->` — Update author info

The conversion engine is in the `<script>` tag at the bottom of `index.html` — look for `FULL_KD_TABLE`.
